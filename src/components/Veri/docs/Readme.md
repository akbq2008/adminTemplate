# 验证码组件

验证码组件

## Attrubutes

| 参数  | 说明                 | 类型   | 可选值 | 默认值     |
| ----- | -------------------- | ------ | ------ | ---------- |
| phone | 接受验证码的手机号码 | String | -      | -          |
| time  | 验证码间隔时间       | Number | -      | 60         |
| txt   | 文案                 | String | -      | 发送验证码 |
